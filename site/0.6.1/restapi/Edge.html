GET api/metadata/lineage/edges/:id

  • Description
  • Parameters
  • Results
  • Examples

Description

Gets the edge with specified id.

Parameters

  • :id is the unique id of the edge.

Results

Edge with the specified id.

Examples

Rest Call

GET http://localhost:15000/api/metadata/lineage/edges/Q6t-c-5g

Result

{
    "results":
        {
            "_id":"Q6t-c-5g",
            "_type":"edge",
            "_outV":12,
            "_inV":16,
            "_label":"output"
        }
}
